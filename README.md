# swamphacks-2021

## Setup

Run the command `npm run init`. YOU MUST INCLUDE THE `run` BEFORE THE INIT

## Running

Use the command `npm start` to launch the server and then navigate to the supplied address

Press `Ctrl+C` to stop the server
